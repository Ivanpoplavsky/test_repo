## 🎯 Исходное Требование

```bash

git clone <repo-url>
cd test_repo

# - Создаст виртуальное окружение
# - Установит зависимости
# - Запустит примеры

```

### Ручное использование

```bash
# 1. Создать виртуальное окружение
python3 -m venv venv

# 2. Активировать его
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запустить примеры
python3 examples.py

# 5. Запустить тесты
python3 -m unittest discover -s tests -p "test_*.py" -v
```

### В коде
```python
from src.currency_converter import CurrencyConverter

converter = CurrencyConverter()

# Конвертировать валюту
result, details = converter.convert_via_crypto(1000, "RUB", "USD")
print(f"1000 RUB = {result:.2f} USD")

# Получить курс обмена
rate = converter.get_exchange_rate("EUR", "JPY")
print(f"1 EUR = {rate:.2f} JPY")

# Красивый вывод
converter.print_conversion(5000, "GBP", "CNY")
```
